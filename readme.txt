== Description ==

Logic Nagoya is a custom WordPress theme designed specifically for Logic Nagoya, a music live house located in Nagoya, Japan. The theme features a modern, dark aesthetic with smooth animations and a focus on showcasing events, gallery items, and venue information.

Features:
* Responsive design for all device sizes
* Custom post types for Events and Gallery
* Customizable sections via WordPress Customizer
* Interactive gallery with modal view
* Event listings with date and details
* Contact form integration (works with Contact Form 7)
* Social media integration

== Installation ==

1. Upload the `logic-nagoya-theme` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure the theme options via the WordPress Customizer

== Frequently Asked Questions ==

= How do I add events? =

Events are added as custom post types. Navigate to Events > Add New in your WordPress admin dashboard and fill in the title, content, featured image, and event date (custom field).

= How do I add gallery items? =

Gallery items are also custom post types. Navigate to Gallery > Add New and add the title, featured image, and any content you want to associate with the gallery item.

= How do I customize the homepage sections? =

All homepage sections can be customized through the WordPress Customizer. Go to Appearance > Customize and you'll find sections for Hero, About, Events, Gallery, Contact, and more.
